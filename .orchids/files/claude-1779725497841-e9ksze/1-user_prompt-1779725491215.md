## ⚙️ FUNCIONALIDADES BACKEND (API)

### ✅ **Lo que SÍ funciona:**

#### 1. **Voice Learning** (`POST /api/voice/learn`)
**Qué hace:**
- Recibe corrección del usuario (original vs corregido)
- Analiza diff (palabras cambiadas, estructura, etc.)
- Extrae regla con Groq LLM
- Guarda corrección en DB
- Actualiza Brand Brain con nueva regla
- Calcula voice match score delta

**Input:**
```json
{
  "draft_id": "uuid",
  "original_text": "Texto original",
  "corrected_text": "Texto corregido"
}
```

**Output:**
```json
{
  "success": true,
  "newRule": {
    "rule": "Usar 'herramientas' en lugar de 'soluciones'",
    "category": "vocabulary",
    "confidence": 85
  },
  "voiceMatchScore": 87,
  "improvement": 2,
  "totalRules": 15
}
```

**Estado:** ✅ **100% funcional**

```json
    }
  },
  "trace_id": "..."
}
```

**Estado:** ✅ **100% funcional**

#### 3. **Agent Review System** (`/api/drafts/[id]/comments`)

**Qué hace:**
- Editor Agent analiza draft automáticamente
- Detecta 5 tipos de problemas:
  - 🎣 **Hook débil** (opening no engaging)
  - 🎤 **Voice mismatch** (tono/vocabulario inconsistente)
  - 🔁 **Repetición** (palabras repetidas >2 veces)
  - 📐 **Estructura** (formato inadecuado para plataforma)
  - 📏 **Longitud** (excede límites de plataforma)
- Genera comentarios con sugerencias accionables
- Guarda en `draft_comments` table
- UI para ver/resolver/filtrar comentarios

**Endpoints:**
- `GET /api/drafts/[id]/comments` - Lista comentarios
- `POST /api/drafts/[id]/comments` - Crea comentario
- `PATCH /api/drafts/[id]/comments/[commentId]` - Resolve/unresolve
- `DELETE /api/drafts/[id]/comments/[commentId]` - Elimina

**Ejemplo de comentario generado:**
```json
{
  "severity": "warning",
  "category": "hook",
  "content": "El hook podría ser más fuerte. Considerá empezar con una pregunta provocativa...",
  "metadata": {
    "score": 35,
    "threshold": 40,
    "suggestions": [
      "Usar una pregunta que genere curiosidad",
      "Incluir un número o estadística sorprendente"
    ]
  }
}
```

**Estado:** ✅ **100% funcional**
